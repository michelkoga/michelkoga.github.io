Switcher (MacOS app)
September 2018 Beta Mostly Working

Switcher is an application for MacOS. Is like the switcher in-app of MacOs, but very powerful. You set the place you want for each app. Then you can call it with your mouse or just press the respective key for each app. That will improve your workflow. Patterns (MacOS app)
April 2018 Working

Patterns is a way to write repetive json code quickly. It was intended to build Karabiner-Elements complex modifications, so I don't know if it would be usable in others situations.

The rule is very simple: Write the code on the left pane. The code that will repeat goes between < and >. The variant part goes inside parenthesis(), separated by comma.
